### 方式 A：Google Calendar (推荐)
1. 复制上面的链接。
2. 打开 [Google日历网页版](https://calendar.google.com/)。
3. 在左侧菜单找到 **"其他日历" (Other calendars)** -> 点击 **"+"** -> **"通过现有网址添加" (From URL)**。
4. 粘贴链接，点击 **"添加日历"**。

### 方式 B：iPhone / Mac (Apple Calendar)
1. 复制上面的链接。
2. **iPhone:** 设置 -> 日历 -> 账户 -> 添加账户 -> 其他 -> 添加已订阅的日历 -> 粘贴链接。
3. **Mac:** 文件 -> 新建日历订阅 -> 粘贴链接。

### 方式 C：Outlook
1. 打开 Outlook 网页版。
2. 添加日历 -> 从 Web 订阅 -> 粘贴链接。

## 🔔 如何开启提醒 (重要)

**Google 日历用户请特别注意：**
由于 Google 的防骚扰机制，订阅的外部日历默认可能**不会**开启通知。如果你希望在关键日期收到提醒，**必须在导入日历后手动开启**：

1. 打开 **Google 日历网页版**。
2. 在左侧找到已添加的 **"港股实战日历 2026"**。
3. 点击旁边的三个点 `⋮` -> **设置和共享 (Settings and sharing)**。
4. 下滑找到 **"活动通知" (Event notifications)**。
5. 点击 **"添加通知" (Add notification)** -> 设置为 **0分钟** (0 minutes before)。

这样，你就会在每个交易日的 **早上 09:00 (香港时间)** 准时收到提醒，不会错过任何一次行情！
