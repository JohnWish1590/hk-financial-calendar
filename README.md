# 🇭🇰 港股关键日历 (2026实战版) / HK Stock Market Key Dates 2026

这个日历整理了 2026 年港股市场的关键交易节点，包括：
- **期指/期权交割日** (Derivatives Expiry)
- **MSCI / 恒生指数 季度调整日** (Index Rebalancing)
- **月末基金调仓日** (Month-End Rebalancing)
- **富时罗素调整日** (FTSE Russell)

特别标注了收盘竞价（U盘）时段可能出现巨量成交的日期。

## 📅 如何订阅 (一键同步)

### 方式 A：Google Calendar (推荐)
1. 复制此链接： https://raw.githubusercontent.com/JohnWish1590/hk-financial-calendar/main/hk_stock_2026.ics
2. 打开 [Google日历网页版](https://calendar.google.com/)。
3. 在左侧菜单找到 "其他日历" (Other calendars) -> 点击 "+" -> "通过现有网址添加" (From URL)。
4. 粘贴链接，点击 "添加日历"。

### 方式 B：iPhone / Mac (Apple Calendar)
1. 复制上面的 `.ics` 链接。
2. iPhone: 设置 -> 日历 -> 账户 -> 添加账户 -> 其他 -> 添加已订阅的日历 -> 粘贴链接。
3. Mac: 文件 -> 新建日历订阅 -> 粘贴链接。

### 方式 C：Outlook
1. 打开 Outlook 网页版。
2. 添加日历 -> 从 Web 订阅 -> 粘贴链接。
