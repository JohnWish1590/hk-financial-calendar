# 港股关键日历 (2026实战版) / HK Stock Market Key Dates 2026

🌟 V2.0 终极版 更新说明 
本次更新基于港交所 2026 年法定休市日历与国际指数公司底层规则进行了深度交叉比对，修复了两个极易导致实战“翻车”的隐蔽陷阱，并核验了二月份的特殊日历规则：

1. 🚨 【日期修正】6月富时罗素（FTSE）调仓提前

原定规则： 富时罗素按惯例在 6 月的第三个星期五（6月19日）生效。

致命陷阱： 2026年 6月19日 恰逢香港端午节，港股全天休市！

V2.0 修正： 根据指数公司“遇节假日提前一交易日”的规则，几十亿外资的被动调仓将提前至 6月18日（星期四） 尾盘进行。日历已将该事件提前，并增加高亮预警。

2. ⏰ 【时间修正】12月31日“半日市”时间陷阱

原定规则： 默认尾盘竞价（U盘）时间为 16:08 - 16:10。

致命陷阱： 12月31日（除夕）在港股属于半日市（Half-Day Trading），下午不交易！如果按常规时间等下午4点看戏，市场早就关门放假了。

V2.0 修正： 将 12月31日“年终基金排名战”的提醒时间和 U盘 时段，精准修正为 中午 12:08 - 12:10。

3. ✅ 【规则核验】2月期指与MSCI背靠背（无须修改，但确实验证无误）

排雷说明： 很多人按“最后一天”的经验，认为 2月28日 是月末。但 28日 是周六，因此最后交易日是 27日（周五）。

结论核实： 2月26日（周四）为期指结算日（倒数第二个交易日），2月27日（周五）为 MSCI 核爆调仓日。原日历此设定完全正确，V2.0 再次确认为安全无误。

这个日历整理了 2026 年港股市场的关键交易节点，包括：
- **期指/期权交割日** (Derivatives Expiry)

## 🔔 如何开启提醒 (重要)

**Google 日历用户请特别注意：**
由于 Google 的防骚扰机制，订阅的外部日历默认可能**不会**开启通知。如果你希望在关键日期收到提醒，**必须在导入日历后手动开启**：

1. 打开 **Google 日历网页版**。
2. 在左侧找到已添加的 **"港股实战日历 2026"**。
3. 点击旁边的三个点 `⋮` -> **设置和共享 (Settings and sharing)**。
4. 下滑找到 **"活动通知" (Event notifications)**。
5. 点击 **"添加通知" (Add notification)** -> 设置为 **0分钟** (0 minutes before)。

这样，你就会在每个交易日的 **早上 09:00 (香港时间)** 准时收到提醒，不会错过任何一次行情！
